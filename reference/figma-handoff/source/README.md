# Original Figma exports

- `page2.svg` is the normative export for the seven Page 2 desktop frames.
- `page1.svg` is preserved as contextual material and must not override the
  Page 2 route and component specifications.

Both files are original Figma exports. They convert all text to paths and
embed raster images, so they are suitable for lossless browser rendering and
Codex visual comparison. They are intentionally not the primary Claude Code
interface; use the normalized JSON and Markdown in the parent directory for
implementation.

Do not edit or optimize these source files in place.
